Теория

Понятие квадратные уравнения, их виды и способы решения

Квадратным уравнением называют алгебраическое уравнение, имеющее вид: ax² + bx + c = 0, где a, b и c – действительные числа, причем a ≠ 0.

Коэффициенты a, b, c различают по названиям:

a - первый или старший коэффициент;

b - второй или коэффициент при х;

c - свободный член, свободен от переменной х.

Существует несколько видов квадратных уравнений.

Приведенное квадратное уравнение - уравнение, где старший коэффициент равен 1:
x² + 2x + 1 = 0

x² + bx + c = 0 - стандартный вид приведенного квадратного уравнения

Неприведенное квадратное уравнение - старший коэффициент отличен от 1:
7x² + 4x + 1 = 0

Полное квадратное уравнение - это квадратное уравнение, в котором присутствуют все три слагаемых; иными словами, это уравнение, у которого коэффициенты b и c отличны от нуля:
2x² - x + 1 = 0

Неполное квадратное уравнение - это уравнение, в котором присутствуют не все три слагаемых; иными словами, это уравнение, у которого хотя бы один из коэффициентов b и c равен нулю:
Примеры квадратных уравнений:

3x²+2x+5=0;

x²+5x=0;

2x²+9=0;

x²+16=0;

x²+3x+4=0

.

Дискриминант

Рассмотрим способы решения таких уравнений.

Основным способом решения является нахождение корней через дискриминант:
D = b² - 4ac

Сами же корни находятся по формулам:

Если D > 0, то уравнение имеет 2 корня:
x₁ = (-b + √D) / 2a     x₂ = (-b - √D) / 2a

Если D = 0, то уравнение имеет 1 корень:
x = -b / 2a

Если D < 0, то уравнение не имеет корней.

Решение квадратного уравнения при чётном коэффициенте b .

Если в квадратном уравнении x²>a+bx+c = 0 второй коэффициент b является чётным, то решение этого уравнения можно немного упростить. Дискриминант для такого уравнения вычисляем по формуле:

D/4 = (b/2)² − a·c

а корни находим по формулам:

x₁ = (-(b/2)² + √(D/4)) / a

x₂ = (-(b/2)² - √(D/4)) / a

Разложение левой части уравнения на множители.

Решим уравнение x² + 10x – 24 = 0

Разложим левую часть уравнения на множители:

x² + 10x – 24 = x² + 12x – 2x – 24 = x(x + 12) – 2(x + 12) = (x + 12)(x – 2)

Следовательно, уравнение можно переписать так:

(x + 12)(x – 2) = 0

Так как произведение равно нулю, то по крайней мере один из его множителей равен нулю. Поэтому левая часть уравнения обращается в нуль при x = 2, а также при x = -12. Это означает, что числа 2 и –12 являются корнями уравнения x² + 10x – 24 = 0

Метод выделения полного квадрата.

Решим уравнение x² + 6x – 7 = 0

Выделим в левой части полный квадрат. Для этого запишем выражение x² + 6x в следующем виде:

x² + 6x = x² + 2·x·3

В полученном выражении первое слагаемое – квадрат числа x, а второе – удвоенное произведение x на 3. Поэтому чтобы получить полный квадрат, нужно прибавить 3^2, так как:

x² + 2·x·3 + 3² = (x + 3)²

Преобразуем теперь левую часть уравнения x² + 6x – 7 = 0 , прибавляя к ней и вычитая 3^2. Имеем:

x² + 6x – 7 = x² + 2·x·3 + 3² – 3² – 7 = (x + 3)² – 9 – 7 = (x + 3)² – 16

Таким образом, данное уравнение можно записать так: (x + 3)² – 16 = 0

Решение уравнений с использованием теоремы Виета для приведённого квадратного уравнения (прямой и обратной)

Формула теоремы Виета для квадратного уравнения (прямая):

Сумма корней квадратного уравнения ax² + bx + c = 0 равна x₁ + x₂ = −b ,

x₁ × x₂ = c.

Если свободный член c приведенного уравнения положителен (c > 0), то уравнение имеет два одинаковых по знаку корня и это зависит от второго коэффициента b.
Если b > 0, то оба корня отрицательные, если b < 0, то оба корня положительны.

Например:
x² – 3x + 2 = 0; x₁ = 2, x₂ = 1 , так как c = 2 > 0, b = –3 <0
x² + 8x + 7 = 0; x₁ = –7, x₂ = –1 , так как c = 7 > 0, b = 8 > 0

Если свободный член c отрицателен (c < 0), то уравнение имеет два различных по знаку корня, и больший по модулю будет положительным, если b < 0, или отрицательным, если b > 0.

Например:
x² + 4x – 5 = 0; x₁ = –5; x₂ = 1 , так как c = –5 < 0, b = 4 > 0
x² – 8x – 9 = 0; x₁ = 9; x₂ = –1 , так как c = –9 < 0, b = –8 > 0

Теорема Виета

Формула теоремы Виета для квадратного уравнения (обратная):

Если числа x₁ и x₂ таковы, что x₁ + x₂ = –b, x₁ × x₂ = c, то они являются корнями квадратного уравнения x² + bx + c = 0.

Решим уравнение x² – 9x + 14 = 0 .
Ищем x₁ и x₂, такие, что x₁ + x₂ = 9, x₁ × x₂ = 14 → x₁ = 2, x₂ = 7.

Решим уравнение x² + 3x – 28 = 0.
Ищем x₁ и x₂, такие, что x₁ + x₂ = –3, x₁ × x₂ = –28 → x₁ = –7, x₂ = 4.
